# 멀티 틱택토 (Multi Ttt)

3개의 말만으로 즐기는 새로운 방식의 틱택토 게임입니다. 4번째 수를 두면 가장 먼저 두었던 말이 사라지는 '선입선출' 규칙이 적용되어, 매 순간 새로운 전략이 필요합니다.

## ✨ 주요 기능

*   **로컬 2인용 모드**: 하나의 PC에서 친구와 함께 즐길 수 있습니다.
*   **AI 대전 모드**: 기본적인 인공지능 상대로 실력을 겨룰 수 있습니다.
*   **온라인 멀티플레이**: 준비된 서버에 접속한 다른 플레이어와 실시간으로 대전할 수 있습니다.

## 룰 설명

1.  각 플레이어(O와 X)는 차례대로 돌아가며 말을 놓습니다.
2.  자신의 말은 최대 3개까지만 보드에 놓을 수 있습니다.
3.  이미 3개의 말이 놓인 상태에서 다음 수를 두면, 자신이 놓았던 가장 오래된 말이 보드에서 사라집니다.
4.  가로, 세로, 대각선 중 하나로 자신의 말 3개를 먼저 정렬하면 승리합니다.

## 🛠️ 사용된 기술

*   **클라이언트 (Client)**
    *   C# (Windows Forms)
    *   SocketIOClient
*   **서버 (Server)**
    *   Node.js
    *   Express
    *   Socket.IO
    *   *현재 클라이언트는 `http://158.179.161.105:3000`에 배포된 서버에 연결됩니다.*

## 🚀 실행 방법

### 터미널에서 실행

아래 명령어를 통해 프로젝트를 실행할 수 있습니다.

```bash
# Client 폴더로 이동
cd Client

# 프로젝트 실행
dotnet run
```

### 직접 실행 (빌드 후)

`dotnet publish` 명령어로 빌드한 후, `Client/bin/Release/net9.0-windows/win-x64/publish/` 와 같은 경로에 생성된 `Multi_Ttt_Client.exe` 파일을 직접 실행할 수 있습니다.
